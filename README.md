## Current features
See the [Roadmap](https://github.com/orgs/Aurora-softwares/projects/1) to know what will be added soon.
* ATA IDE / AHCI Driver.
* FAT32/16/12 + Virtual FileSystem.
* PCI Device Scan.
* PS/2 Keyboard + Mouse.
* Restart + ACPI Shutdown.
* Command interpreter.
* Exception Handler.
* GUI (SVGAII, VBE)
* Networking (ARP, IPV4, ICMP, UDP, DNS, DHCP, TCP, HTTP)
